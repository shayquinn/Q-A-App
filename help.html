Q&A App File Format Guide

This application is designed to parse HTML exports from KeepNote pages. An example file named Example.html is included in the project folder for reference.

To upload a new quiz, your HTML file should follow this structure:

1. Questions Section

The app supports two formats for questions. You can use whichever is easier to create.

Format A (Simple / Raw)

This format is common when exporting from word processors. The question number is bolded, followed by the text, and options are in lists below it.

1. Question text goes here?
  • Option A
  • Option B

Format B (Structured)

In this format, the question number is inside the list item.

  • 1. Question text here?
    • Option A
    • Option B

2. Answers Section

The answers section should be at the bottom of the file, marked by "Answers".

Answers
  • 1. A. Explanation text...
  • 2. B. Explanation text...
